# baoma-inc/.github

This repository is the special [GitHub Organization Profile](https://docs.github.com/en/organizations/collaborating-with-groups-in-organizations/customizing-your-organizations-profile) repository for **baoma-inc**.

The content in [profile/README.md](profile/README.md) is displayed publicly on the main organization landing page: [https://github.com/baoma-inc](https://github.com/baoma-inc).

## Repository Structure

```text
.github/
├── README.md               # This file (meta documentation)
├── profile/
│   └── README.md           # The main organization landing page profile
└── .gitignore              # Standard gitignore configurations
```

## How to Update the Profile

1. Clone this repository locally:
   ```bash
   git clone https://github.com/baoma-inc/.github.git
   ```
2. Modify the file [profile/README.md](profile/README.md).
3. Commit and push your changes to the `main` branch.
4. Verify the updates on the organization profile page: [github.com/baoma-inc](https://github.com/baoma-inc).

## Contact & Security

If you find any security vulnerabilities or credentials leaked in this profile, please immediately contact the security team or open an issue on the private internal trackers. Do not post credentials here.
